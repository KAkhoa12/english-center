...

Xác thực bằng JWT
- Đăng nhập hệ thống -> cho access_token -> Đi kiểm tra xem có hợp lệ hay không"
+ access_token hợp lệ -> cho truy vấn api :
	* Nếu giải mã access_token cho user_id ( ví dụ id = 1 ): gọi api cập nhập thông tin cá nhân user id = 1 -> thì nó sẽ cho cập nhập
	* Nếu giải mã access_token cho user_id ( ví dụ id = 1 ): gọi api cập nhập thông tin cá nhân user id = 2 -> thì nó sẽ không cho cập nhập
+ access_token k hợp lệ -> trả về lỗi 401 unauthorize

Cơ chế access_token chỉ tồn tại trong 30p, refesh_token tồn tại trong 30 ngày

====================== CÁCH CHẠY DỰ ÁN =====================

Thư mục dự án: https://github.com/KAkhoa12/english-center.git

B1: Vào trong thư mục trên máy tạo 1 folder lưu trữ code của mình
B2: clone src này về bằng lệnh git:  git clone https://github.com/KAkhoa12/english-center.git
B3: cd english-center

== Chạy src FE trước  ==
bắt buộc có: nodejs, pnpm

1. cd english-center-client
2. pnpm i
3. pnpm dev

Không cần chạy docker == Chạy docker compose ==  ( Chạy 1 lần duy nhất cho lần đầu )
bắt buộc có: docker destop
1. về lại thư mục gốc english-center
2. cd english-center-backend
3. docker compose up -d
Hiển thị chấm xanh là thành công

== Chạy src BE ===
options: dbeaver, hoặc 1 ide hỗ trợ kết nối PostgreDB, thư viên uv, python >=3.12
1. Kết nối DB
2. Chuột phải commections -> create -> connection -> PostgresSQL ( Chạy 1 lần duy nhất cho lần đầu )
3. Chỉnh các thông số như sau: ( Chạy 1 lần duy nhất cho lần đầu )
- database name: english_center_db
- username: postgres
- password: postgres
4. Nhấn Test connection hiển thị ok thì đã kết nối thành công  ( Chạy 1 lần duy nhất cho lần đầu )
5. cp .env.local > .env   ( Chạy 1 lần duy nhất cho lần đầu )
6. pip install uv ( Nếu như chưa cài ) ( Chạy 1 lần duy nhất cho lần đầu )
7. uv sync ( tải tất cả các thư viện ) ( Chạy 1 lần duy nhất cho lần đầu )
8. .venv\Scripts\activate.bat, nếu dùng vs code tạo 1 terminal git bash: source .venv/Scripts/activate
9. hiển thị biến môi trường ảo như (english-center-backend) là thành công
10. python -m alembic upgrade head  ( Chạy 1 lần duy nhất cho lần đầu )
11. uvicorn app.main:app --reload
